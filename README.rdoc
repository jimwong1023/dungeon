== Dungeon Maze

CONRTROLS 
- Move forward (up arrow), right (right arrow), left (left arrow), back (back arrow).
- Move upstairs (w), downstairs (s). 

This "Dungeon Maze" was created using three.js to render each room. 

The most difficult aspect of the application was keeping track of direction and rendering the individual walls and doors at the correct locations according to your current point of view.
